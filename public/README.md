# Movix

**Movix** es una plataforma de movilidad urbana que muestra vehículos, locales, buses y servicios en tiempo real sobre un mapa interactivo.

## Características
- Mapa LeafletJS con marcadores conectados a Firebase.
- Registro/Login de usuario.
- Visualización por capas: autos, colectivos, servicios, buses, locales.
- Modo oscuro y claro.
- Entrada a red social tipo comunidad.
- Selección de zona para negocios desde el mapa.
- Preparado para despliegue en Netlify con carpeta `public/`.

## Estructura
```
public/
├── index.html
README.md
```

## Despliegue
1. Subir a un repositorio GitHub.
2. Conectar a Netlify y seleccionar `/public` como carpeta de publicación.
