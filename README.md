# Movix 🌍🚗  
Mapa interactivo para gestión de locales comerciales y aforo en tiempo real.

## Tecnologías  
- **Frontend**: HTML5, LeafletJS, Firebase.  
- **Backend**: Firebase Realtime Database.  

## Cómo Usar  
1. Clona el repositorio.  
2. Abre `public/index.html` en tu navegador.
